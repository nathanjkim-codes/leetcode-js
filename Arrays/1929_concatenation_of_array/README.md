# 1929. Concatenation of Array

## Problem

Given an integer array `nums`, create a new array by concatenating `nums` with itself.

Example:

Input:

nums = [1,2,1]

Output:

[1,2,1,1,2,1]

---

## My Solution

- Used two `for` loops.
- Created a new array called `ans`.
- Added each element using `push()`.
- Returned the final array.

---

## What I Learned

- How to iterate through an array using a `for` loop.
- The difference between `nums` and `nums[i]`.
- How `push()` adds elements to the end of an array.
- Why we create a new array instead of modifying the original one.
